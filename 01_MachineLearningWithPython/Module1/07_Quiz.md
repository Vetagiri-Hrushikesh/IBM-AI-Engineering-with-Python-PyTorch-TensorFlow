## Machine Learning and AI – Self-Study Quiz

> **Instructions:**  
> For each question, select the most suitable answer. After choosing, check the explanation to understand your choice.

***

**Question 1:**  
While studying customer purchases in a supermarket, you find that “customers who buy bread often buy butter.” Which machine learning technique suits this task?

A) Classification  
B) Clustering  
C) Regression  
D) Association

**Correct Answer:** D) Association  
> *Explanation:* Association techniques find items/events that occur together often, like shopping basket analysis.

***

**Question 2:**  
Which workflow correctly describes the lifecycle of a machine learning project?

A) Prepare model → Clean model → Visualize model → Report model  
B) Prepare data → Design UI → Train model→ Collect feedback  
C) Define problem → Collect data → Prepare data → Develop and evaluate model → Deploy model  
D) Define problem → Prepare data → Deploy model

**Correct Answer:** C) Define problem → Collect data → Prepare data → Develop and evaluate model → Deploy model  
> *Explanation:* The full ML model lifecycle involves stating the problem, gathering and preparing data, developing and testing models, then deploying them.

***

**Question 3:**  
You are combining user behavior logs from different sources into a central table. What process does this represent?

A) Transforming data before training the model  
B) Providing recommendations  
C) Building a dashboard  
D) Preparing PowerPoint slides

**Correct Answer:** A) Transforming data before training the model  
> *Explanation:* Aggregating and cleaning data is an essential “transform” step before model training.

***

**Question 4:**  
You want to build and export a supervised learning model in Python. Which tool should you use?

A) Python  
B) Pandas  
C) Scikit-learn  
D) Matplotlib

**Correct Answer:** C) Scikit-learn  
> *Explanation:* Scikit-learn is a key Python library for training, evaluating, and exporting ML models.

***

**Question 5:**  
Which statement correctly describes Scikit-learn’s capabilities?

A) Generates AI art and images before data preprocessing  
B) Works with unstructured video and audio data  
C) Preprocesses data, trains, evaluates, and exports ML models  
D) Only plots and graphs for visualization

**Correct Answer:** C) Preprocesses data, trains, evaluates, and exports ML models  
> *Explanation:* Scikit-learn supports the full end-to-end ML workflow.

***

**Question 6:**  
Which programming language is most popular for machine learning due to its libraries and ease of use?

A) JavaScript  
B) Java  
C) Python  
D) Scala

**Correct Answer:** C) Python  
> *Explanation:* Python dominates ML due to libraries like NumPy, Pandas, Scikit-learn, TensorFlow, etc.

***

**Question 7:**  
Which Python library provides the foundation for fast numerical computations on big multi-dimensional arrays (used in ML)?

A) NumPy  
B) Scikit-learn  
C) Pandas  
D) SciPy

**Correct Answer:** A) NumPy  
> *Explanation:* NumPy is the basis for efficient math in Python and forms the core of scientific/ML computing.

***

**Question 8:**  
What type of machine learning is used when you train on labeled data to predict new outcomes?

A) Supervised Learning  
B) Unsupervised Learning  
C) Semi-supervised Learning  
D) Reinforcement Learning

**Correct Answer:** A) Supervised Learning  
> *Explanation:* Supervised learning uses labeled data for prediction.

***

**Question 9:**  
Which tool would you pick to visualize complex data using layered graphics in R?

A) Matplotlib  
B) ggplot2  
C) Seaborn  
D) Tableau

**Correct Answer:** B) ggplot2  
> *Explanation:* ggplot2 lets you build layered visualizations in R.

***

**Question 10:**  
What ML technique is used for predicting continuous numbers, like house prices?

A) Clustering  
B) Classification  
C) Regression  
D) Association

**Correct Answer:** C) Regression  
> *Explanation:* Regression predicts numerical (continuous) values, like prices or quantities.
